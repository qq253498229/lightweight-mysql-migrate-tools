== 使用方法

引入jar包

[source,xml,subs="verbatim,attributes"]
----
<dependency>
  <groupId>cn.codeforfun</groupId>
  <artifactId>mysql-migrate</artifactId>
  <version>{version}</version>
</dependency>
----

测试类

[source,java]
----
include::{testdir}/cn/codeforfun/migrate/core/MigrateTest.java[]
----
